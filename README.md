# Smart CV 🚀

An intelligent resume builder that helps you create, format, and optimize your CV automatically.

## ✨ Features
* **Automated Parsing:** Import data directly from text or LinkedIn profiles.
* **Modern Themes:** Choose from multiple clean, professional templates.
* **Export Options:** Download your CV instantly as a polished PDF.

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone 
   cd smart-cv
   ```

2. **Install dependencies:**
   ```bash
   npm install  # or pip install -r requirements.txt
   ```

3. **Run the project:**
   ```bash
   python manage.py runserver
   ```

## ⚙️ Configuration
Modify the `config.json`and '.env' file to update your personal information, contact details, and work history.
